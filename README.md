# 台大電機之夜 2026 幸運抽獎系統

🎉 台大電機之夜專屬抽獎網站

## 功能特色

- ✨ 賽博朋克視覺風格設計
- 🎯 防重複抽獎機制
- 📊 即時統計與進度追蹤
- 🎁 支援 9 種獎品類型
- 👥 588 位參與者（觀眾編號 + 具名參與者）
- 📋 完整名單查看與篩選功能

## 獎品列表

1. Miss game 跳動格子單人免費券（每人2張）- 抽10位
2. 瑪琍歐保時捷遙控車 - 1位
3. 旅行快煮壺 - 2位
4. 米家調理機 - 1位
5. ADATA 無線磁吸式行動電源 - 2位
6. 三麗鷗旅行掛扣帶 - 50位
7. 味全龍智勝引退球衣手燈 - 10位
8. 龍將心願斗方 - 10位
9. 小龍女DB Show鑰匙圈（隨機不挑款）- 20位

## 使用方式

1. 點選左側獎品列表中的任一獎品
2. 點擊「開始抽獎」進行滾動抽獎
3. 看到中獎者後，點擊「確認中獎」正式記錄
4. 系統自動準備下一次抽獎
5. 可隨時點擊「查看抽獎名單」查看所有參與者

## 技術特點

- 純前端實現，無需後端伺服器
- 響應式設計，支援各種裝置
- 動態粒子背景效果
- 流暢的動畫與過渡效果

## 部署

使用 GitHub Pages 即可輕鬆部署：
1. 將此專案 push 到 GitHub
2. 在 Settings > Pages 中啟用 GitHub Pages
3. 選擇 main 分支作為來源
4. 網站將自動部署
